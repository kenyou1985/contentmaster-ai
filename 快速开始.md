# 🚀 快速开始指南

## ✅ 已完成
- ✅ 依赖已安装完成
- ✅ Node.js 版本: v22.19.0
- ✅ npm 版本: 10.9.3

## 🎯 下一步操作

### 方式 1: 使用启动脚本（推荐）
双击运行 `启动项目.bat` 文件，即可自动启动开发服务器。

### 方式 2: 手动启动
在项目目录打开终端，运行：
```bash
npm run dev
```

## 🔑 配置 API Key

启动应用后，有两种方式配置 Gemini API Key：

### 方式 1: 在应用界面输入（最简单）
1. 打开浏览器访问 http://localhost:3000
2. 在界面顶部找到 API Key 输入框
3. 输入你的 Gemini API Key
4. 系统会自动保存到本地

### 方式 2: 使用环境变量
1. 在项目根目录创建 `.env.local` 文件
2. 添加以下内容：
   ```
   VITE_GEMINI_API_KEY=你的_API_Key
   ```
3. 重启开发服务器

## 📝 获取 API Key

如果你还没有 Gemini API Key，可以：
1. 访问 https://ai.google.dev/
2. 注册/登录 Google 账号
3. 创建 API Key

## 🌐 访问应用

启动成功后，在浏览器中打开：
```
http://localhost:3000
```

## ⚠️ 注意事项

- 如果端口 3000 被占用，可以修改 `vite.config.ts` 中的端口号
- API Key 会保存在浏览器的 localStorage 中
- 如果使用代理服务（如云雾 AI），API Key 以 `sk-` 开头时会自动切换
