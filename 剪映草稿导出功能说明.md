# 剪映草稿导出功能

## 快速开始

### macOS（必须两步）

**第一步：启动剪映导出服务**（单独终端窗口运行）

```bash
# 方式一：双击运行（推荐）
./启动剪映导出服务.command

# 方式二：终端运行
chmod +x 启动剪映导出服务.sh
./启动剪映导出服务.sh
```

**第二步：启动主应用**

```bash
npm run dev
```

然后访问 http://localhost:3000 ，在「媒体生成」页面的操作栏右侧找到 **「导出剪映草稿」** 按钮。

---

### Windows

安装 pyJianYingDraft 后，直接启动主应用即可：

```bash
pip install pyJianYingDraft
npm run dev
```

---

## 工作原理

```
前端 (React)
    ↓ fetch('/api/jianying/...')
Node.js HTTP 服务 (server/server.mjs, 端口 18091)
    ↓ child_process.spawn('python3')
Python 脚本 (services/jianying_export_service.py)
    ↓
macOS: 在 ~/Movies/JianyingPro/User Data/Projects/com.lveditor.draft/ 创建草稿文件夹 + 打开剪映
Windows: 使用 pyJianYingDraft.JianyingController 自动化导出视频
```

## 功能说明

| 平台 | 自动生成草稿 | 自动导出视频 | 依赖 |
|------|------------|------------|------|
| macOS | ✅ | ❌（需手动在剪映点击导出） | Python 3 |
| Windows | ✅ | ✅ | pyJianYingDraft + 剪映专业版 5.9-6.x |

## API 端点

- `GET /api/jianying/health` — 健康检查
- `GET /api/jianying/list` — 列出所有草稿
- `POST /api/jianying/export` — 导出草稿

## 注意事项

- **macOS**：草稿生成后，需要在剪映中手动打开该草稿 → 点击导出按钮
- **剪映版本**：建议使用剪映专业版 5.9（支持草稿格式最完整），6+ 版本已加密草稿文件
- **素材路径**：图片需为本地路径（http URL 会被跳过，v2 可扩展为自动下载）
